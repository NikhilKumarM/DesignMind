# DesignMind

A Chrome extension that replaces your new tab with daily system design quotes, tips, and challenges — built for developers who want to level up every time they open a tab.

## Features

- **Daily Quote** — A deterministic system design quote that changes each day
- **Daily Tip** — A practical system design tip that rotates daily
- **Daily Challenge** — A system design problem with a hint and a reveal-able answer
- **Google Search Bar** — Centered search input; type a query to Google it or enter a URL to navigate directly
- **Daily Streak Counter** — Tracks consecutive days of usage with a fire badge (glows at 7+ day milestones)
- **Copy Quote** — One-click copy to clipboard with confirmation
- **Shuffle Quote** — Load a random quote on demand
- **Greeting + Live Clock** — Time-of-day greeting with emoji and a live updating clock
- **Keyboard Shortcuts** — Press `/` to focus the search bar, `Escape` to blur it
- **Offline-First** — All data is bundled locally, no network needed

## Tech Stack

- HTML, CSS, JavaScript (vanilla)
- Chrome Manifest V3
- No external dependencies (fonts loaded from Google Fonts)
